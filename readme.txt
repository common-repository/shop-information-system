== Description ==
* Author: [Fahad Mahmood](https://www.androidbubbles.com/contact)
* Demo URI: <http://demo.androidbubble.com/please-wait/>

Shop Information System facilitates price adjustment of products in bulk as per given values. You can increase/decrease price as flat rate and discount based. You can also select products, price and other parameters to see sales history on graphs. You can normalize price as per previous price history and reset price to last updated price, you can go back to the last entry in price history.


== Overview ==
* Price adjustment of products in bulk
* Increase prices as percentage/flat rate
* Decrease prices as percentage/flat rate
* Shop analysis > Product sales, Product prices
* Normalize price as per previous history

== Frequently Asked Questions ==

= How to adjust prices of products using this plugin? =
Select "Adjustment" then select adjustment by flat/percentage. Select increase/decrease, enter price/percentage and Update.

= What is normalize button do? =
Normalize button allow you to set average prices as per pervious history.

= How to reset prices and how many times it could be? =
Select "Reset" and press update, system will undo price to last price, it can undo till last entry in price history.

= What if I decrease price and one or more products price might become 0 or less? =
In this case you will see warning as you enter amount in Price Adjustment box even if you do not select any product.

= If one or more products are on sale already what will happen after changing price? =
One or more products are on sale and new price is greater than or equal to regular price, so products are removed from sale and use regular price for future.


= Uninstalling Shop Information System =

When you uninstall the plugin, files will still remain in your WordPress library because these files are linked within these directories not physically moved within.

== Installation ==

From the WordPress plugin menu click on Add New and search for Shop Information System

Instead of searching for a plugin you can directly upload the plugin zip file by clicking on Upload:

Use the browse button to select the plugin zip file that was downloaded, then click on Install Now. The plugin will be uploaded to your blog and installed. It can then be activated.

"Shop Information System" is available under WooCommerce.
